Negrito / Destaque

Nesta linha de parágrado temos o o termo em destaque usando a tag <b>.

Ítalico / Ênfase

Neste parágrado teremos o termo em destaque usando a tag <i>, sem significado para apresentar efeitos de formatação.

Nesta linha de parágrado, teremos o termo em destaque usando a tag <em> embutido de significado desde o seu termo.

Marcadores, textos deletados e inseridos.

Nesta linha de parágrado teremos o termo em destaque marcado usando a tag <mark>

Nesta linha de parágrado teremos o termo deletado usando a tag <strike>.

Nesta linha teremos a tag <del>, para que ele delete essa parte.

Nesta linha de parágrado teremos o termo nesta formatação para ser lido com mais atenção.

Sobrescrito e Subescrito

Nesta linha teremos as tags <sub> e a tag <sob>.
